# Twilight Agent Skills

This repo contains Claude Code skills for autonomous trading on Twilight Protocol.

## Skills

- **`/twilight-trader`** — Wallet management + trade execution via `relayer-cli`
- **`/twilight-strategies`** — Query the Strategy API for live hedging/arb opportunities

## How to use

1. Clone this repo into your project or add as a submodule
2. Invoke `/twilight-trader` to trade or `/twilight-strategies` to analyze strategies
3. See `README.md` for full API reference and endpoint details
